## Graphics APIs

- [DirectX 11 (Windows 7+)](https://docs.microsoft.com/en-us/windows/desktop/direct3d11/atoc-dx-graphics-direct3d-11)
- [DirectX 12 (Windows 10+)](https://docs.microsoft.com/en-us/windows/desktop/direct3d12/direct3d-12-graphics)
- [OpenGL 4.6](https://www.khronos.org/opengl/)
- [Vulkan](https://www.khronos.org/vulkan/)

## Dependencies

- [The Open-Asset-Importer-Lib (Assimp)](http://www.assimp.org/)
- [The OpenGL Extension Wrangler Library (GLEW)](http://glew.sourceforge.net/)
- [OpenGL Mathematics (GLM)](https://glm.g-truc.net/)
- [OpenGL Registry (wglext.h)](https://www.khronos.org/registry/OpenGL/index_gl.php)
- [A tiny JSON library for C++11](https://github.com/dropbox/json11)
- [libnoise](http://libnoise.sourceforge.net/)
- [LZMA SDK](https://www.7-zip.org/sdk.html)
- [wxWidgets](https://www.wxwidgets.org/)
